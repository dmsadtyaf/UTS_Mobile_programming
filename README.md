FitMotion App adalah Aplikasi untuk membantu jadwal workout anda, melihat banyak orang yang sangat malas berolahraga dikarenakan tidak tau akan olahraga apa yang harus dilakukan hari ini, dengan FitMotion kamu tidak akan bingung lagi, tinggal ikuti jadwal yang sudah tersedia, lalu centang jika sudah dilakukan, dan selamat kamu sudah membakar kalori kamu
Fitur yang terdapat di FitMotion App adalah Fitur Login, Lupa Password, Fitur Logout, Fitur Progress berapa persen sesuai workout yang dicentang
Cara Menjalankan Aplikasi, menggunakan mobile phone API di Vs Code, setelah emulator terbuka, lalu Run Main.dart
